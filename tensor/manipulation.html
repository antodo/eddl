Rearranging elements and transformations¶

shift¶

void Tensor::shift(Tensor *A, Tensor *B, vector<int> shift, WrappingMode mode = WrappingMode::Constant, float cval = 0.0f)¶

Shift the tensor. The array is shifted using spline interpolation. Points outside the boundaries of the input are filled according to the given mode.

Parameters

  • A: Input tensor.

  • B: Output tensor.

  • shift: vector of shifts along the axes.

  • mode: Must be one of the following:

    • WrappingMode::Constant: Input extended by the value in cval (v v v v | a b c d | v v v v)

    • WrappingMode::Reflect: Input extended by reflecting about the edge of the last pixel (d c b a | a b c d | d c b a)

    • WrappingMode::Nearest: Input extended by replicating the last pixel (a a a a | a b c d | d d d d)

    • WrappingMode::Mirror: Input extended by reflecting about the center of the las pixel (d c b | a b c d | c b a)

    • WrappingMode::Wrap: Input extended by wrapping around the oposite edge (a b c d | a b c d | a b c d)

    • WrappingMode::Original: Input extended by placing the original image in the background.

  • cval: Value to fill past edges of input if mode is WrappingMode::Constant 

string fname = "../../examples/data/elephant.jpg";  // Some image
Tensor* t1 = Tensor::load(fname);
Tensor* t2 = new Tensor(t1->shape);

Tensor::shift(t1, t2, {50, 100}, WrappingMode::Constant, 0.0f); // Shifts t1 50 pixels in y and 100 in x.

void Tensor::rand_uniform(float v)¶

Generates uniformly distributed random samples inplace.

Parameters

  • v: Scale factor of the values generated by the uniform distribution. 

Tensor* t1 = Tensor::ones({3});
t1->rand_uniform(1.0);//Fills t1 with samples from a uniform distribution.

rand_normal¶

void Tensor::rand_normal(float m, float s, bool fast_math = true)¶

Generates normal distributed random samples inplace.

Parameters

  • m: Mean of the normal distribution.

  • s: Standard deviation of the normal distribution.

  • fast_math: Wether to use or not the fast math mode. 

Tensor* t1 = Tensor::ones({3});
t1->rand_normal(0, 1); //Fills t1 with samples from a normal distribution with mean 0 and std 1.
