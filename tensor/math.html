Abs¶

Tensor *Tensor::abs()

Element-wise abs operation.

Return

A new tensor with abs applied over A

// [
// [0.31 0.08 1.59]
// [-0.77 -0.46 -0.39]
// ]

t1->abs_();  // In-place
// [
// [0.31 0.08 1.59]
// [0.77 0.46 0.39]
// ]

// Other ways
Tensor* t2 = t1->abs(); // returns a new tensor
Tensor::abs(t1, t2); // static
