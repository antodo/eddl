All Close?¶

bool Tensor::allclose(Tensor *A, Tensor *B, float rtol = 1e-05, float atol = 1e-08, bool equal_nan = false)¶

Returns True if two arrays accomplish, element-wise, the condition \(|A-B| \leq atol+rtol\times|B|\).

Return

void

Parameters

  • A: Input tensor.

  • B: Input tensor.

  • rtol: relative tolerance.

  • atol: absolute tolerance.

  • equal_nan: if True, then two NaNs will be considered equal. 

Tensor* t1 = Tensor::range(1, 6); t1->reshape_({2,3});
t1->print(2);  // Temp.
// [
// [1.00 2.00 3.00 4.00 5.00]
// [6.00 7.00 8.00 9.00 10.00]
// [11.00 12.00 13.00 14.00 15.00]
// [16.00 17.00 18.00 19.00 20.00]
// [21.00 22.00 23.00 24.00 25.00]
// ]


Tensor* t2 = Tensor::range(1, 6); t2->reshape_({2,3});
t2->print(2);  // Temp.
// [
// [1.00 2.00 3.00 4.00 5.00]
// [6.00 7.00 8.00 9.00 10.00]
// [11.00 12.00 13.00 14.00 15.00]
// [16.00 17.00 18.00 19.00 20.00]
// [21.00 22.00 23.00 24.00 25.00]
