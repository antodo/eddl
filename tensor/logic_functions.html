Comparison¶

allclose¶

bool Tensor::allclose(Tensor *A, Tensor *B, float rtol = 1e-05, float atol = 1e-08, bool equal_nan = false)¶

Returns True if two arrays accomplish, element-wise, the condition \(|A-B| \leq atol+rtol\times|B|\).

Return

void

Parameters

  • A: Input tensor.

  • B: Input tensor.

  • rtol: relative tolerance.

  • atol: absolute tolerance.

  • equal_nan: if True, then two NaNs will be considered equal. 

static bool allclose(Tensor *A, Tensor *B, float rtol=1e-05, float atol=1e-08, bool equal_nan=false);  // Returns true or false

isclose¶

void Tensor::isclose(Tensor *A, Tensor *B, Tensor *C, float rtol = 1e-05, float atol = 1e-08, bool equal_nan = false)¶

Returns a boolean array where a position is true if elements in A and B accomplish \(|A-B| \leq atol+rtol\times|B|\).

Return

void

Parameters

  • A: Input tensor.

  • B: Input tensor.

  • C: Output tensor.

  • rtol: relative tolerance.

  • atol: absolute tolerance.

  • equal_nan: if True, then two NaNs will be considered equal. 

static void isclose(Tensor *A, Tensor *B, Tensor *C, float rtol=1e-05, float atol=1e-08, bool equal_nan=false);  // Returns a boolean tensor
